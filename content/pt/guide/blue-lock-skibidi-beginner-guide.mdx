<Section icon={Target} title="Dominando Controles e Atalhos">

Para ter sucesso nas partidas, você deve fixar os controles de computador na memória muscular. Embora os movimentos específicos dos personagens sejam mais chamativos, a movimentação padrão, os pedidos de passe e o posicionamento defensivo é que vencem os jogos. 

A tabela abaixo descreve todos os atalhos essenciais para computador que você precisa para sobreviver às suas primeiras partidas.

| Ação | Tecla de Atalho | Função Principal | Dica de Tempo de Partida |
|---|---|---|---|
| **Chutar / Afastar** | Clique Esquerdo | Chuta a bola para frente | Fique de frente para o seu alvo; use na área para marcar ou afastar |
| **Carrinho** | E | Defesa com carrinho | Ative depois que o atacante escolher uma direção |
| **Drible** | Q | Controle de bola padrão | Ative quando um defensor se aproximar para passar por ele |
| **Salto do Goleiro** | Q | Defesa do goleiro | Espere a bola ser chutada; não pule cedo demais |
| **Pedir Passe** | R | Sinalize para os companheiros de equipe | Pressione quando encontrar espaço aberto no terço de ataque |
| **Habilidades Especiais** | 1, 2, 3, 4, 5 | Movimentos do personagem | Verifique os requisitos da habilidade antes de ativar |

<Section icon={Shield} title="Fundamentos para Goleiros">

Jogar como Goleiro (GK) é uma das funções mais impactantes do jogo. Um goleiro habilidoso pode parar atacantes lendários e garantir vitórias fáceis para sua equipe.

Para jogar com sucesso no gol, equipe um personagem defensivo como **Gagamaru** e siga estas regras críticas de posicionamento.

| Regra de Goleiro | Ação | Benefício Estratégico |
|---|---|---|
| **Fique Centralizado** | Posicione-se perto da linha do gol | Mantém os dois cantos da trave ao alcance de um salto |
| **Atrase seu Salto** | Pressione **Q** apenas após o chute | Evita que os atacantes provoquem seu salto antes do tempo e entrem com a bola no gol |
| **Afaste para as Laterais** | Fique de frente para as linhas laterais antes de chutar | Evita passar a bola diretamente para os meio-campistas adversários |

<Section icon={Trophy} title="Coordenando Combos de Equipe">

Conforme você avança, jogadas individuais não serão suficientes para quebrar defesas organizadas. Você deve utilizar reações químicas e jogadas coordenadas em equipe para marcar contra goleiros de alto nível.

O combo de equipe mais famoso é a **Reação Química de Reo e Nagi**, que permite que sua equipe passe pelos defensores instantaneamente.

<CardGrid cols={3}>
  <Card title="1. Posicionamento">
    O jogador com o Nagi deve se mover para o espaço aberto à frente do jogador com o Reo, rompendo a linha defensiva.
  </Card>
  <Card title="2. O Passe">
    O jogador com o Reo deve ficar de frente para o Nagi e executar um passe direto usando os controles de chute padrão.
  </Card>
  <Card title="3. A Reação">
    Assim que o passe chegar, o Nagi ativa a reação química para executar um chute subsequente de alta velocidade.
  </Card>
</CardGrid>

<Callout type="warning" title="Falhas no Combo">
Este combo falhará se ambos os jogadores estiverem usando o mesmo personagem, se a linha de passe estiver bloqueada por um defensor, ou se o chute subsequente for atrasado.
</Callout>

</Section>
